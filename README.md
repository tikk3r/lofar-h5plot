# lofar-h5plot
Inspect an H5Parm with

    python h5plot.py <h5parm>

# Requirements
* Python 3
* LoSoTo 2.0
* PyQt5
* Matplotlib

These can be installed through

    pip install pyqt5 matplotlib
    pip install --upgrade https://github.com/revoltek/losoto/archive/master.zip
